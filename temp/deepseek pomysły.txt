🎯 PROJEKTY ŚREDNIEJ ZŁOŻONOŚCI
1. GameMatch AI - Inteligentne dopasowanie gier

LLM analizuje opisy gier z RAWG i dopasowuje je do:
- Twojego nastroju ("mam ochotę na coś relaksującego")
- Okazji ("gram z przyjaciółmi którzy nigdy nie grali")
- Czasu ("mam tylko 30 minut")
- Sprzętu ("gram na słabym laptopie")

Innowacja: Personalizacja wykraczająca poza tagi
2. Narrative Weaver - Generator fabuł/fan fiction

LLM tworzy alternatywne zakończenia/scenariusze 
bazując na opisie gry z RAWG

Przykład: "Co by było, gdyby Geralt w Wiedźminie 3 
podjął inne decyzje? Stwórz alternatywną historię"

4. Accessibility Advisor

LLM analizuje:
- Recenzje z RAWG
- Opisy gier
- Opinie graczy

I sugeruje modyfikacje dla graczy:
- z dysleksją (czcionki, tempo tekstu)
- z problemami wzroku (kontrasty)
- jednorękich (remapowanie klawiszy)

🛠️ KONKRETNE IMPLEMENTACJE
Szkielet FastAPI endpointu:

@app.post("/analyze-game")
async def analyze_game(game_id: int, prompt_template: str):
    # 1. Pobierz dane z RAWG
    game_data = await get_rawg_data(game_id)
    
    # 2. Przygotuj kontekst dla LLM
    context = f"""
    Tytuł: {game_data['name']}
    Opis: {game_data['description']}
    Gatunki: {', '.join(game_data['genres'])}
    Ocena: {game_data['rating']}
    """
    
    # 3. Wywołaj LLM (OpenAI/Anthropic/Local)
    llm_response = await call_llm(
        system_prompt="Jesteś ekspertem od gier...",
        user_prompt=f"{prompt_template}\n\n{context}"
    )
    
    return {
        "game_info": game_data,
        "ai_analysis": llm_response,
        "suggestions": extract_suggestions(llm_response)
    }
